 # Road-ing

## 「Road-ing」是一款提供地圖顯示、路線規劃和用戶互動的露營Android應用程式。

## 功能
- **營區資料**：顯示並排名(根據用戶評論之情緒分析之綜合評分)指定條件篩選之營區。
- **用戶註冊**：創建和管理用戶資料。
- **路線規劃**：規劃路線並顯示地圖項目。
- **每日登入coupon**：提供登入coupon使用於抵免金額。
